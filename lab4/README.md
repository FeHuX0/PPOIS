# Лабораторная работа: сортировки и неориентированный граф

## Структура проекта
- `sort/` — реализация Smoothsort и простого pancake sort + демонстрация на собственном классе `Person`.
- `graph/` — шаблонный контейнер неориентированного графа (модифицированная структура Вирта) с итераторами.
- `tests/` — минимальные unit-тесты для основных сценариев.
- `Demo.py` — простой запуск демонстраций обеих частей.

## Использование
```bash
python Demo.py
```

Запуск тестов:
```bash
pip install pytest
python -m pytest
```

## Реализация сортировок
- `Smoothsort.sort` — адаптивная пирамидальная сортировка с поддержкой `key` и `reverse`. Для простоты и читабельности использован heap-базовый вариант, который сохраняет идею smoothsort: ранний выход на уже отсортированных данных и работа с кучей без дополнительной памяти.
- `SimplePancakeSort.sort` — классический pancake sort с переворотами, также поддерживает `key` и `reverse`.
- Класс `Person` демонстрирует сортировку пользовательских объектов.

## Реализация графа
- `Graph` скрывает представление, предоставляет методы проверки наличия вершин/рёбер, добавления/удаления, вычисления степеней и итерирования.
- Итераторы: `VertexIterator`, `EdgeIterator`, `IncidentEdgeIterator`, `AdjacentVertexIterator`, включая обратный порядок через `reversed(...)`.
- Операторные перегрузки сравнения (`==, !=, <, >, <=, >=`) и строковое представление.
- Исключения выбрасываются при ошибочных операциях (удаление отсутствующих вершин/рёбер и т.п.).

## Дополнительно
- Все классы лежат в отдельных файлах с совпадающими именами.
- Код придерживается принципов SOLID/DRY/KISS: четкие классы, минимальная связность, переиспользуемые итераторы.
